# Manual Test Case Scenarios

## Scope
These manual test cases validate lift movement, door sequence, 5-second door-open hold, and floor light behavior defined in the instruction.

## Preconditions
- System has 2 floors: floor 1 and floor 2.
- Each floor has exactly one active light at any time: RED, AMBER, or GREEN.
- Timer measurement method is available (stopwatch or log timestamps) for the 5-second door-open hold.

## Test Cases

| Test Case ID                            | Action                                                                                                                                                      | Expected Result                                                                                                                                                     |
|-----------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| TC-01 Idle Red                          | Set lift to idle parking mode with no active calls.                                                                                                        | RED is ON for the evaluated floor. AMBER and GREEN are OFF.                                                                                                       |
| TC-02 Running Car Call Red              | Trigger car call service state (isRunningForCarCall = true).                                                                                               | RED is ON while running for car call. AMBER and GREEN are OFF unless higher-priority conditions become true.                                                     |
| TC-03 Loaded State Red                  | Set loaded state (isLoadedState = true) with no landing-run or door-opened condition.                                                                     | RED is ON. AMBER and GREEN are OFF.                                                                                                                                |
| TC-04 Door Opening Car Red              | Set door opening for car call (isDoorOpeningForCarCall = true).                                                                                            | RED is ON during car-call door opening.                                                                                                                            |
| TC-05 Running Landing Call Amber        | Trigger landing call service state (isRunningForLandingCall = true).                                                                                       | AMBER is ON while running for landing call. RED and GREEN are OFF unless GREEN condition becomes true.                                                            |
| TC-06 Door Opening Landing Amber        | Set door opening for landing call (isDoorOpeningForLandingCall = true).                                                                                    | AMBER is ON during landing-call door opening.                                                                                                                      |
| TC-07 Door Opened Landing Green         | Set door opened for landing call (isDoorOpenedForLandingCall = true).                                                                                      | GREEN is ON. RED and AMBER are OFF due to priority.                                                                                                                |
| TC-08 Door Opened Car Green             | Set door opened for car call (isDoorOpenedForCarCall = true).                                                                                              | GREEN is ON. RED and AMBER are OFF due to priority.                                                                                                                |
| TC-09 Priority Green Over Amber and Red | Set isDoorOpenedForCarCall = true and also set Red/Amber triggers (for example isRunningForCarCall = true and isRunningForLandingCall = true).          | GREEN remains the only active light.                                                                                                                               |
| TC-10 Priority Amber Over Red           | Set isRunningForLandingCall = true and isRunningForCarCall = true while both door-opened states are false.                                               | AMBER remains the only active light.                                                                                                                               |
| TC-11 Movement To Other Floor (Landing Call) | Set currentFloor = 1 and targetFloor = 2. Request landing call for floor 2. Execute sequence.                                                        | Lift starts moving toward floor 2. At floor 2, sequence is: door opening -> door opened -> stays opened for 5 seconds -> door closing -> door closed.           |
| TC-12 Same Floor Service (Landing Call) | Set currentFloor = 2 and targetFloor = 2. Request landing call for floor 2. Execute sequence.                                                             | Lift does not move. At floor 2, sequence is: door opening -> door opened -> stays opened for 5 seconds -> door closing -> door closed.                          |
| TC-13 Movement To Other Floor (Car Call) | Set currentFloor = 1 and targetFloor = 2. Request car call for floor 2. Execute sequence.                                                                 | Lift starts moving toward floor 2. At floor 2, sequence is: door opening -> door opened -> stays opened for 5 seconds -> door closing -> door closed.           |
| TC-14 Same Floor Service (Car Call)     | Set currentFloor = 2 and targetFloor = 2. Request car call for floor 2. Execute sequence.                                                                 | Lift does not move. At floor 2, sequence is: door opening -> door opened -> stays opened for 5 seconds -> door closing -> door closed.                          |
| TC-15 Door Hold Duration                | Trigger a call and observe the moment door reaches opened state until closing starts.                                                                      | Elapsed opened duration is 5 seconds (+/- acceptable system tolerance).                                                                                            |
| TC-16 Single Active Light Contract      | For each of the above states, observe light output per floor.                                                                                              | Exactly one light is active per floor at any instant. Never more than one active color on the same floor.                                                        |

## Notes For Test Execution
- Run each case for both floors where relevant.
- Record timestamped evidence for movement start, door opened, and door closing to verify timing requirements.
- If a case fails, capture state input values and resulting output light for root-cause analysis.
